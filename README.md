# SortVis

This is an experimental project I made because I had a dumb idea of how to visualize sorting functions in the terminal in C (though this has definitely been done before but who cares).

Currently all in one file; I'll clean up the architecture later (maybe, hopefully). Currently bubble sort, insertion sort, selection sort, merge sort, and supported. If anyone finds this and wants to add another sorting algorithm it should be easy. The only thing to know is where to put the Visualize function
